Multi Keyword Ranked Search



A basic service provided in cloud computing is to outsource the data to extrinsic cloud servers.

As data remains on cloud servers, it is necessary to encrypt the outsourced data to preserve the secrecy and confidentiality concerns of owner of data as outsource data may contain sensitive privacy information, such as personal photos, emails.

This encryption process may raise some complexities on the precise search of data over the encrypted cloud data. As a solution of this problem we focus on multiple keywords that is entered in search query by data user and give him most relevant data by using tf-idf technique .
